# PsevdoCognetive - Когнитивные тренировки

Веб-приложение для когнитивных тренировок мозга.

## Тренировки

| Название | Описание |
|----------|----------|
| Игра на память | Запоминание и повторение подсвеченных плиток на сетке |
| Память с вращением | Запоминание плиток с последующим вращением поля 1-3 раза |
| Блоки Корси | Запоминание последовательности подсвечивания (прямой/обратный порядок) |
| Таблица Шульте | Поиск чисел от 1 до 25 на время |
| Тест Струпа | Определение соответствия названия цвета и цвета текста за 30 секунд |
