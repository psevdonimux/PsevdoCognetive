# PsevdoCognetive - Cognitive Training

Web application for brain cognitive training.

## Trainings

| Name | Description |
|------|-------------|
| Memory Game | Memorize and repeat highlighted tiles on a grid |
| Rotation Memory | Memorize tiles with subsequent field rotation 1-3 times |
| Corsi Blocks | Memorize highlighting sequence (forward/reverse order) |
| Schulte Table | Find numbers from 1 to 25 against time |
| Stroop Test | Determine if color name matches text color in 30 seconds |
